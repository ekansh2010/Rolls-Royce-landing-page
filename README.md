# 🏎️ Rolls-Royce Landing Page

A visually rich and immersive landing page inspired by the elegance and luxury of Rolls-Royce. This project focuses on smooth animations, cinematic video backgrounds, and premium UI design to deliver a high-end user experience.

---

## ✨ Features

* 🎬 Full-screen cinematic video backgrounds
* 🧭 Interactive navigation menu with overlay panel
* 🖱️ Smooth scrolling and hover animations
* 🚗 Dynamic car showcase sections
* 🎨 Premium dark-themed UI inspired by luxury branding
* 📱 Responsive and modern layout

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the webpage 
* **CSS3** – Styling, animations, and layout design 
* **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior

## 🚀 Key Functionalities

### 🎥 Video Experience

* Auto-playing, looping background videos for each section
* Smooth transitions between luxury car showcases

### 📜 Scroll Effects

* Logo changes dynamically on scroll
* Fade-in animation for sections using Intersection Observer

### 📂 Navigation System

* Sliding overlay menu
* Smooth anchor-based scrolling

### 🎯 UI Interactions

* Hover animations on buttons
* Image zoom effects on car cards
* Animated arrow transitions
